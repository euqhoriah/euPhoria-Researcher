📚 What is this?
This is an educational proxy project to demonstrate how web proxies work. Enter any URL below to see how proxy servers forward requests and responses.

How This Works

This educational project demonstrates proxy concepts:

  • Request Forwarding: Your request is sent through a proxy server
  • Protocol Handling: Supports both HTTP and HTTPS
  • Response Forwarding: The response is sent back to you
  • Header Management: Proper HTTP header handling
  • Error Handling: Graceful handling of connection issues

⚠️ Note: This is for educational purposes only. Some websites may not work perfectly due to CORS (Cross-Origin Resource Sharing) restrictions or script blocking.
